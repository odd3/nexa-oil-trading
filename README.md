# Nexa Oil — Corporate Website

A premium, fully-responsive corporate website for a Dubai-based international energy
trading company (crude oil & industrial lubricants). Static HTML/CSS/JS — no build
step or dependencies required to run.

## Open it

Just double-click **`index.html`**, or serve the folder:

```bash
cd desert-prime-trading
python3 -m http.server 8000
# visit http://localhost:8000
```

## Languages 🇬🇧 🇩🇪

The site is fully bilingual **English + German**. English pages live at the root,
German pages under **`de/`**. Every page has an **EN | DE** switch in the header (and in
the mobile menu) that links to the same page in the other language. Navigation, all body
content, forms, footer and SEO meta are translated. `<html lang>` and `hreflang`
alternate tags are set per page for search engines.

- English home: `index.html`
- German home: `de/index.html`

## Design

- **Colours:** navy `#071528`, gold `#D4AF37`, sand `#E8D8B0`, white, grey.
- **Fonts:** Playfair Display (headings) + Inter (body), via Google Fonts.
- Sticky header, scroll-reveal animations, animated counters, hero video-style
  imagery (Unsplash CDN), custom SVG world map, mobile slide-out menu, accordions,
  and mock-validated forms.
- SEO meta tags, Open Graph, semantic markup, responsive (mobile-first).

## Editing content

The pages are generated by `build.py` (shared shell + EN/DE menu strings) plus
`build_pages.py` (English content) and `build_pages_de.py` (German content). Edit the
content there and run `python3 build.py` to regenerate all 24 HTML files. You can also
edit the `.html` files directly — they are self-contained. Shared styling lives in
`assets/css/style.css` and behaviour in `assets/js/main.js`. To translate into a further
language, copy `build_pages_de.py` and add the language to the `T` dictionary in `build.py`.

## Notes

- The Request-a-Quote and Contact forms are front-end demos (`data-mock`); wire them to
  an email service or backend (e.g. Formspree, a serverless function, or your CRM) to go live.
- Replace the Unsplash hero/section images in `build.py` (the `IMG` dictionary) with your
  own licensed photography before production launch.
- Update the contact email `sales@nexaoiltrading.com` and add real address/phone details.
